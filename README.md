# SpinJam - Juggling Event Website

A single-page informational website for an upcoming juggling event, built with SvelteKit.

## Getting Started

Install dependencies:

```sh
npm install
```

## Development

Start the development server:

```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Project Structure

- `src/routes/+page.svelte` - Main page content
- `src/routes/+layout.svelte` - Layout wrapper with metadata
- `static/` - Static assets

> To deploy your app, you may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
